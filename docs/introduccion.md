# Introducción

Las redes informáticas permiten que dispositivos como ordenadores, móviles o servidores
puedan comunicarse entre sí para compartir información.

En esta guía aprenderás:

- Cómo se estructuran las redes.
- Cómo funciona el direccionamiento IP.
- Qué son los modelos OSI y TCP/IP.
- Cómo realizar configuraciones básicas.
- Herramientas comunes de diagnóstico.

---

## ¿Qué es una red?

Una red es un conjunto de dispositivos interconectados que pueden comunicarse
entre ellos mediante reglas llamadas *protocolos*.

---

## Tipos de redes

- **LAN**: redes locales (oficinas, casas).
- **WAN**: redes extensas (Internet).
- **WLAN**: redes inalámbricas.

> **Dato:** Internet es una red WAN que conecta millones de redes LAN.
