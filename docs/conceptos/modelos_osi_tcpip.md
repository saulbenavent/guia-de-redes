# Modelos OSI y TCP/IP

Los modelos OSI y TCP/IP definen cómo se comunican los dispositivos en una red.

---

# Modelo OSI (7 capas)

1. **Física**  
2. **Enlace de datos**  
3. **Red**  
4. **Transporte**  
5. **Sesión**  
6. **Presentación**  
7. **Aplicación**

---

### Ejemplo práctico

Cuando abres una web:

- La capa 7 envía la petición (HTTP).
- La capa 4 usa TCP.
- La capa 3 calcula rutas (IP).
- La capa 1 envía bits por el cable o WiFi.

---

# Modelo TCP/IP (4 capas)

1. **Acceso a la red**  
2. **Internet**  
3. **Transporte**  
4. **Aplicación**

---

## Comparativa

| OSI | TCP/IP |
|-----|--------|
| 7. Aplicación | 4. Aplicación |
| 4. Transporte | 3. Transporte |
| 3. Red | 2. Internet |
| 1–2 Físico y Enlace | 1. Acceso a la red |

---

> *El modelo OSI es teórico. TCP/IP es el modelo real que usan las redes hoy.*
